IOT-Enabled Smart Irrigation System for theAgricultural Sector

Apr 2025 - Present

Project Description

Developed a smart irrigation system using ESP32 microcontrollers, soil moisture sensors, and cloud-based automation. Implemented real-time sensor data monitoring and automated water delivery based on soil conditions, weather forecasts, and crop needs. Designed an interactive dashboard using Python, JavaScript, and MQTT for live monitoring. Deployed the system with AWS IoT and Proxmox for device management and edge computing. This project aimed to optimize water usage and enhance agricultural productivity through smart technology.
